# 드래곤 나이트 - 게임 멘탈 모델

## 핵심 루프

```
[탐색/이동] → [전투 진입] → [실시간 액션 + ATB 축적] → [스킬 발동 (슬로모션)] → [전투 해결] → [탐색/이동]
```

## 전투 타임라인

게임은 **두 가지 시간 흐름**이 공존한다:

```
실시간 레이어 (TimeScale = 1.0)
├── 플레이어 이동 (좌/우/점프/대시)
├── 기본 공격 (근접, 실시간)
├── 적 AI (배회 → 인식 → 공격 패턴)
└── ATB 게이지 자동 충전 ■■□□ (2칸)

        ↓ ATB 게이지 소모하여 스킬 발동

슬로모션 레이어 (TimeScale ≈ 0.1)
├── 스킬 리스트 패널 표시 (6개)
├── 선택한 스킬의 범위 원 표시
│   └── 보너스 방향 강조 (상/하/좌/우 중 1)
├── 범위 내 대상 선택
└── 스킬 이펙트 → TimeScale 복귀
```


## 방향 보너스 메커니즘

스킬마다 보너스 방향이 하나 지정됨. 범위 원 안에서 해당 방향이 강조 표시된다:

```
        [상 보너스]
            ▲
            │
  [좌]◄─── ● ───►[우]     ● = 스킬 시전 지점
            │
            ▼
        [하 보너스]

예) "화염참" 스킬 - 보너스: 우측
    → 적이 시전자 우측에 있을 때 적중하면 추가 효과
```

## 전투 시퀀스 예시

```
1. 플레이어가 좌우로 이동하며 기본공격으로 적과 교전
2. 시간 경과 → ATB ■■ (2칸 충전됨)
3. 스킬 버튼 입력 → TimeScale 감소 (슬로모션)
4. 스킬 리스트 표시 → "화염참" 선택 (1칸 소모)
5. 범위 원 + 우측 보너스 강조 표시
6. 원 안의 적 선택 → 해당 적이 우측에 있으므로 보너스 적중!
7. 이펙트 재생 → TimeScale 복귀
8. ATB ■□ (1칸 남음) → 전투 계속
```

## 동료 소환 흐름

```
소환 입력 → 동료 등장 애니메이션 → 특수 능력 사용 → 퇴장 → 쿨타임 시작
                                                              │
                                                    쿨타임 완료 시 재소환 가능
```

## 핵심 설계 포인트

**TimeScale 전환**이 이 게임의 가장 핵심적인 설계 포인트다.
실시간 액션과 ATB 스킬 선택이 같은 씬에서 시간 스케일만 바꿔가며 공존하기 때문에,
모든 게임플레이 시스템(적 AI, 물리, 애니메이션)이 `Time.timeScale`에 올바르게 반응해야 한다.
UI는 `Time.unscaledDeltaTime`으로 동작시켜야 슬로모션 중에도 정상 조작이 가능하다.
