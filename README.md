# 드래곤 나이트

## 프로젝트 목적

언리얼 엔진의 사용법과 횡스크롤 실시간 전투 게임 개발을 공부하기 위한 프로젝트

## 게임 컨셉

- 주인공은 여신의 선택을 받은 용사
- 마왕을 쓰러뜨려 세계를 악에서 구해야 한다
- 여정 중에 많은 동료들을 만난다.

## 게임 시스템

- 횡스크롤 액션
- 플레이어는 근접 기본 공격을 실시간으로 사용
- ATB 시스템으로 스킬 사용
- 시간에 따라 ATB게이지 상승
- ATB게이지는 2칸
- 게이지를 사용해서 스킬 사용
- 스킬은 6개 제한
- 스킬 실행을 입력하면 스킬 리스트 패널이 켜지고 전투 시간이 느려짐
- 스킬마다 정해진 범위가 원으로 표시
- 원 안에서 스킬을 사용할 대상을 선택 가능
- 스킬마다 정해진 방향에서 적중시 보너스 효과
- 정해진 방향도 원 안에 표시
- 동료를 2명 설정해 둘 수 있음
- 동료를 소환하면 동료가 나타나서 특수 기술사용하고 퇴장
- 동료마다 쿨타임 있음
